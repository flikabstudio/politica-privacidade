Política de Privacidade

Última atualização: 05/05/2026

Esta Política de Privacidade descreve como as informações são coletadas, usadas e protegidas no aplicativo BÍBLIA E HARPA cRISTÃ.

1. Informações Coletadas

O aplicativo pode coletar automaticamente informações como tipo de dispositivo, sistema operacional, identificadores únicos e dados de uso.

2. Uso das Informações

As informações são usadas para melhorar a experiência do usuário e exibir anúncios relevantes.

3. Publicidade

Este aplicativo utiliza o Google AdMob para exibir anúncios, podendo coletar o ID de publicidade do dispositivo.

4. Compartilhamento

Não vendemos dados pessoais. Informações podem ser compartilhadas com serviços de terceiros confiáveis.

5. Segurança

Adotamos medidas para proteger seus dados, mas nenhum sistema é totalmente seguro.
